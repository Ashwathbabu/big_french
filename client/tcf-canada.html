Side by Side

TCF vs TEF: Quick Comparison

Both exams are accepted by IRCC. Hover each row to learn more.

About the Names
Both are official French proficiency tests recognized for Canadian immigration. Names reflect their administering organizations and testing approach.
Administering Bodies
TCF is run by France Éducation International (a French government body). TEF is managed by CCI Paris Île-de-France, a chamber of commerce. Both are fully IRCC accredited.
Immigration Recognized
Both TCF Canada and TEF Canada are fully accepted by IRCC for Express Entry, PNP, and family sponsorship applications.
Listening Test
TCF is faster with fewer questions — great if you prefer shorter tests. TEF has more exposure with 60 questions over 40 minutes.
Which Should You Choose?
TCF is ideal if you want a quicker exam. TEF Canada is better if you prefer more question exposure — and it's available right now!
